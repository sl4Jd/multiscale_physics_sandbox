# BUILD
### clone the repo
```bash
git clone https://github.com/sl4Jd/game_engine.git
```
```bash
cd game_engine
```
### Install deps
```bash
vcpkg install
```
### Build deps
```bash
cmake -B build -S .
```
### Build app
```bash
cmake --build build
```
